# AI 交互记录脱敏说明

## 必须脱敏的内容

提交前，任何原始 AI 交互记录都必须删除或替换以下内容：

- sudo / 登录 / 远程服务密码；
- SSH 私钥、公钥以外的认证材料、API key、token、cookie；
- 个人身份证明、手机号、邮箱等隐私信息；
- 与比赛无关的本机用户名、绝对路径中可识别个人身份的部分；
- 临时服务地址、端口、内部网络信息，除非它们是复现实验所必需且不敏感；
- 大量无意义的编译日志、依赖下载日志、重复命令输出。

## 建议替换格式

| 原始内容类型 | 替换示例 |
| --- | --- |
| 明文密码 | `<REDACTED_PASSWORD>` |
| API key / token | `<REDACTED_TOKEN>` |
| 私钥 | `<REDACTED_PRIVATE_KEY>` |
| 个人主目录 | `<USER_HOME>` |
| 临时绝对路径 | `<REPO_ROOT>` 或 `<TMP_DIR>` |
