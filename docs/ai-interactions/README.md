# AI 交互记录归档

> 本目录依据《2026 年全国大学生计算机系统能力大赛操作系统设计赛技术方案》1.1 节第 5 点建立，用于归档 Veriskein 开发过程中与 AI 工具 / AI Agent 的关键交互记录。

## 归档原则

- 本目录保存的是**脱敏后的摘要式记录**，不是逐字聊天全文。
- 摘要保留：时间、工具、任务目标、用户请求、AI 辅助内容、执行过的关键命令、验证结果、人工确认事项。
- 摘要删除或替换：密码、密钥、token、个人隐私、无关命令噪声、临时环境细节。
- AI 产生的建议均需经过参赛队人工审阅、测试或修改后才进入最终提交。
